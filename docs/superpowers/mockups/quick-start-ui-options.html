Quick start UI options

Pick the layout that feels best for starting a single-round game from favorite courses. Each option keeps the same behavior: choose a favorite course, choose players, start the game with course and tees resolved automatically.

Tee default rule: for each selected player, use their last-used tee on that course. If the player has no history there, use the course's middle tee.
